# YoutubeMp3Downloader
This is a very simple Windows application created for "quick and easy" YoutTube audio download.<br />
It downloads and automatically converts audio to mp3 file.<br />
You can also download video in 1080p and highest bitrate audio. If there is no 1080p, it downloads video in highest quality possible.<br />
Default download path is desktop - application was created for my parents so they can worry about nothing :)<br />
Conversion is possbile thanks to [FFmpeg](https://ffmpeg.org/). Ensure that the FFmpeg binary is located in your application's working directory.

Libraries:
* [YoutubeExplode](https://github.com/Tyrrrz/YoutubeExplode)
* [FFmpeg](https://ffmpeg.org/)
